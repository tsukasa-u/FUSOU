# Getting Started Snapshot

- Pull the latest changes from `main`
- Install dependencies with `pnpm install`
- Duplicate `.env.example` to `.env`
- Run `pnpm --filter FUSOU-APP dev`
- Confirm the docs site responds on port 4321

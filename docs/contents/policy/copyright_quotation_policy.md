# 著作権と引用に関する運用方針

## 1. 基本方針

本サービスで使用する画像・データは、著作権法第32条に基づく引用の範囲で取り扱います。
著作権その他の権利は、各権利者に帰属します。

## 2. 表示・利用の原則

- 表示目的は、研究・検証・情報提供の範囲に限定します。
- 引用対象は、必要最小限の範囲で利用します。
- 出典や文脈を損なう編集・改変は行いません。
- 第三者による転載・再配布を推奨しません。

## 3. 権利者からの申告対応

権利者または代理人から、削除依頼・警告・権利侵害申告を受けた場合は、内容確認のうえ速やかに次のいずれかを実施します。

- 該当コンテンツの非公開化
- 該当コンテンツの修正
- 必要な補足説明の追加

## 4. 連絡窓口

削除申請は、公開の Issue または非公開のメール連絡で受け付けます。

- 公開窓口: GitHub Issues
- 非公開窓口: ogu.hide.u [at] gmail [dot] com

公開窓口で個人情報の記載が難しい場合は、メール連絡を優先してください。

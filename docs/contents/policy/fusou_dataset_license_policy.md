# FUSOU-DATASET ライセンスと利用方針

## 1. 対象

この文書は、次の 2 つを区別して扱います。

- SDK（fusou-datasets ライブラリ）
- データセット本体（配信されるデータ）

## 2. SDK のライセンス

`packages/fusou-datasets/python/pyproject.toml` の定義に基づき、fusou-datasets Python SDK は MIT ライセンスで提供します。

## 3. データセット本体の取り扱い

データセット本体の権利は、SDK のソースコードライセンスとは別に管理します。
運用上の原則は以下のとおりです。

- データセットの権利は各権利者に帰属します。
- 利用者は、法令および公開される利用条件に従って利用します。
- 商用利用・再配布・二次利用の可否は、個別の利用条件に従います。

## 4. 利用制限と問い合わせ

利用条件に不明点がある場合、または商用利用・再配布を検討する場合は、事前に運営へ確認してください。

- 連絡先: ogu.hide.u [at] gmail [dot] com

## 5. 免責

本方針は運用指針であり、個別事案における法的判断を代替するものではありません。
必要に応じて専門家へ相談してください。

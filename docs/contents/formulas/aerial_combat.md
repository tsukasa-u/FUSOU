## 1. 制空値計算

### 1-1. 基本制空値

```
スロット制空値 = floor((対空値 + 改修補正) x sqrt(搭載数) + 熟練度ボーナス)
```
- **対空値**: 装備の対空ステータス（艦娘の対空ではない）
- **搭載数**: スロットの現在搭載数
- 艦隊全体の制空値 = 全艦の全スロット制空値の合計
- **出典**: 全サイト一致

### 1-2. 改修による制空値ボーナス

| 装備種 | 改修係数 | 適用方法 | 出典 |
|--------|---------|---------|------|
| 艦上戦闘機・水上戦闘機・陸上戦闘機 | 0.2 x ★ | 対空値に加算（線形） | JP Wiki/KCWIKI |
| 艦上爆撃機 | 0.25 x ★ | 対空値に加算（線形） | JP Wiki/KCWIKI |

### 1-3. 熟練度ボーナス

熟練度ボーナスは2要素から成る:
```
熟練度ボーナス = sqrt(内部熟練度 / 10) + 熟練度帯ボーナス
```

#### 内部熟練度（0～120）

| 表示 | 内部熟練度 | sqrt(内部/10)の概算 |
|------|-----------|------------------|
| なし | 0～9 | 0～0.94 |
| \| | 10～24 | 1.0～1.54 |
| \|\| | 25～39 | 1.58～1.97 |
| \|\|\| | 40～54 | 2.0～2.32 |
| //// | 55～69 | 2.34～2.62 |
| \\\\ | 70～84 | 2.64～2.89 |
| \\\\\ | 85～99 | 2.91～3.14 |
| >> (MAX) | 100～120 | 3.16～3.46 |

#### 熟練度帯ボーナス（航空機種別）

| 熟練度帯 | 艦戦・水戦 | 水上爆撃機 | その他(艦攻,艦爆) | 出典 |
|---------|-----------|-----------|-----------------|------|
| 0 | 0 | 0 | 0 | 全サイト |
| 1 | 0 | 0 | 0 | 全サイト |
| 2 | 2 | 1 | 1 | 全サイト |
| 3 | 5 | 1 | 1 | 全サイト |
| 4 | 9 | 1 | 1 | 全サイト |
| 5 | 14 | 3 | 1 | 全サイト |
| 6 | 14 | 3 | 1 | 全サイト |
| 7 (MAX) | **22** | **6** | **1** | 全サイト |

### 1-4. 制空状態の判定

| 制空状態 | 条件 | 出典 |
|---------|------|------|
| 制空権確保 (AS+) | 自軍 >= 敵軍 x 3 | 全サイト一致 |
| 航空優勢 (AS) | 自軍 >= 敵軍 x 1.5 | 全サイト一致 |
| 航空均衡 (AP) | 上記いずれでもない | 全サイト一致 |
| 航空劣勢 (AD) | 自軍 x 3 <= 敵軍 | 全サイト一致 |
| 制空権喪失 (AI) | 自軍 = 0 又は条件外 | 全サイト一致 |

### 1-5. 基地航空隊の制空値（偵察機補正）

```
補正後制空値 = floor(基地航空隊の合計制空値 x 偵察機補正係数)
```

| 偵察機種別 | 出撃時補正 | 防空時補正 | 出典 |
|-----------|-----------|-----------|------|
| 陸上偵察機（偵察値 >= 9） | 1.18 | 1.24 | JP Wiki/KCWIKI |
| 陸上偵察機（偵察値 = 8） | 1.15 | 1.21 | JP Wiki/KCWIKI |
| 大型飛行艇 | 1.12 | 1.16 | JP Wiki/KCWIKI |
| その他 | 1.0 | 1.0 | 全サイト |

**差異**: KCWIKI は一部の特殊陸偵（二式陸上偵察機(熟練)等）に **x1.24** を記載する場合あり

## 2. 航空戦 Stage1・Stage2

### 2-1. Stage1（制空戦・相互撃墜）

各スロットの被撃墜数:
```
被撃墜数 = floor(搭載数 x 撃墜割合)
撃墜割合 = random(下限, 上限)  <- 一様乱数
```

| 制空状態 | 自軍被撃墜率 | 敵軍被撃墜率 | 出典 |
|---------|-------------|-------------|------|
| 制空権確保 | 1%～10% | 7%～65% | 全サイト一致 |
| 航空優勢 | 4%～20% | 20%～45% | 全サイト一致 |
| 航空均衡 | 3%～60% | 3%～60% | 全サイト一致 |
| 航空劣勢 | 20%～45% | 4%～20% | 全サイト一致 |
| 制空権喪失 | 7%～65% | 1%～10% | 全サイト一致 |

### 2-2. 触接

#### 触接開始率
```
触接開始率 = Sigma(0.04 x 装備索敵値 x sqrt(搭載数))  ※搭載数>0のスロットのみ
```

#### 触接選択判定
```
触接選択率 = 制空定数 / (0.1 x 装備命中値 + 1)
```

| 制空状態 | 制空定数 |
|---------|---------|
| 制空権確保 | 3 |
| 航空優勢 | 2 |
| 航空均衡以下 | 1 |

#### 触接攻撃力倍率

| 触接機の命中値 | 倍率 | 出典 |
|--------------|------|------|
| 命中 +3 以上 | x1.20 | 全サイト一致 |
| 命中 +2 | x1.17 | 全サイト一致 |
| 命中 +1 以下 | x1.12 | 全サイト一致 |

## 3. 対空砲火（Stage2）

### 3-1. 総撃墜数

```
Stage2撃墜数 = 割合撃墜 + 固定撃墜 + 対空CI保証撃墜
```

### 3-2. 割合撃墜

```
割合撃墜 = floor(floor(加重対空値 x AACI変動倍率) x 連合補正 / 400 x 現在搭載数)
```
- 対空CIが未発動の場合、AACI変動倍率 = 1.0

### 3-3. 固定撃墜

```
固定撃墜 = floor((floor(加重対空値 x AACI変動倍率) + floor(艦隊防空値 x AACI変動倍率)) x 連合補正 x K値 / 10)
```

### 3-4. 対空CI保証撃墜

```
保証撃墜 = floor(AACI固定ボーナス x AACI変動倍率) + 固定追加B
```
- AACI未発動時: 味方=固定A:1, 固定B:0 / 敵=固定A:0, 固定B:0
- 最低保証: Stage2で対空砲火が行われた場合、最低1機の撃墜

### 3-5. 連合艦隊補正（Mod_CF）

| 状況 | 補正値 |
|------|--------|
| 通常vs通常 | 1.0 |
| 通常vs連合 | 0.8 |
| 連合主力（水上） | 0.8 |
| 連合主力（空襲） | 0.72 |
| 連合護衛 | 0.48 |

## 4. 加重対空値（個艦防空）

```
加重対空値 = floor(素対空 / 2) + Sigma(装備対空値 x 装備倍率 + 改修補正)
-> 最終値 = 2 x floor(加重対空値 / 2)  ※偶数に切り捨て
```

| 高射装置 | **3** | 全サイト一致 |

## 5. 艦隊防空値

```
艦隊防空値 = (2/1.3) x floor(陣形補正 x Sigma floor(Sigma(装備対空値 x 艦隊防空倍率 + 改修補正)))
```

### 6-2. 発動率について

- 全サイトで **「検証中」「推定」** の扱い
- 確定した公式は存在しない
- 複数のAACIを同時に満たす場合、**優先度の高いものから判定**
- KCWIKI が個別のAACIごとの推定発動率を数値で記載

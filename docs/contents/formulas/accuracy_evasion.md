## 1. 命中率計算

### 1-1. 最終命中率

```
命中率 = floor(Hit_cap) + 熟練度命中補正 + 1
```

キャップ付き命中率:
```
Hit_cap = MIN(MAX((命中項 - 回避項), 10) x 防御側士気補正, 96)
```
- **実効範囲**: 最低11% ～ 最高97%
- **出典**: KCWIKI

### 1-2. 命中項（Accuracy Term）

```
命中項 = floor(基本命中 x 陣形命中補正 + 装備命中 + 改修命中補正) + 1
```

基本命中:
```
基本命中 = 90 + 2 x sqrt(Level) + 攻撃側の運（微小影響）
```
- **出典**: KCWIKI/Fandom が `2 x sqrt(Level)` を記載。JP Wiki でも同等

## 2. 回避計算

### 2-1. 回避項

```
回避項(pre-cap) = floor((回避値 + sqrt(2 x 運)) x 陣形回避補正)
```

### 2-2. 回避キャップ処理

| 条件 | 処理 | 出典 |
|------|------|------|
| <= 40 | そのまま | KCWIKI |
| 40～65 | floor(40 + 3 x sqrt(pre-cap - 40)) | KCWIKI |
| >= 65 | floor(55 + 2 x sqrt(pre-cap - 65)) | KCWIKI |

### 2-3. 陣形回避補正

| 陣形 | 回避補正 | 出典 |
|------|---------|------|
| 単縦陣 | 1.0 | 全サイト |

| 輪形陣 | 1.1 | 全サイト |
| 梯形陣 | 1.2 | 全サイト |
| 単横陣 | 1.3 | 全サイト |

## 3. クリティカル

### 3-1. クリティカル率

```
クリティカル率 = floor(Mod_type x sqrt(Hit_cap + 熟練度命中補正)) + 熟練度クリティカル補正 + 1
```

### 3-2. 種別定数（Mod_type）

| 攻撃種別 | 定数 | 出典 |
|---------|------|------|
| 砲撃 | 1.3 | KCWIKI |
| 対潜・雷撃 | 1.5 | KCWIKI |
| 航空攻撃 | 0 | KCWIKI |
| 夜戦 | 1.5 + 夜偵定数 | KCWIKI |

### 3-3. クリティカルダメージ

```
クリティカルダメージ = floor(攻撃力 x 1.5 x 熟練度クリティカル補正)
```
- 空母の搭載スロット熟練度が影響（第1スロットの寄与が最大）

## 4. 索敵計算（33式 / Formula 33）

### 4-1. 完全な計算式

```
索敵スコア = Sigma(全艦) sqrt(素索敵)
           + Cn x Sigma(全装備) (Ce x (装備索敵値 + C★ x sqrt(★)))
           - ceil(Q x 司令部レベル)
           + 2 x M
```

### 4-2. パラメータ

- **Cn**: 分岐点係数（海域・分岐点ごとに異なる）
- **Ce**: 装備カテゴリ係数
- **C★**: 改修索敵係数
- **Q**: 司令部レベル係数
- **M**: 出撃艦数補正

### 4-3. 装備カテゴリ係数

| 装備カテゴリ | Ce | C★ | 出典 |
|-------------|----|----|------|
| 水上偵察機 | **1.2** | 1.2 | 全サイト一致 |
| 水上爆撃機 | **1.1** | 1.1 | 全サイト一致 |
| 艦上偵察機 | **1.0** | 1.2 | 全サイト一致 |
| 艦上攻撃機 | **0.8** | 0 | 全サイト一致 |
| 艦上爆撃機 | - | - | 全サイト |
| 大型飛行艇 | **0.6** | 1.2 | 全サイト一致 |
| 小型電探 | **1.25** | - | 全サイト一致 |
| 大型電探 | **1.4** | - | 全サイト一致 |
| 連絡機 | **1.0** | - | KCWIKI |
| その他（索敵値あり） | **0.6** | 0 | 全サイト |
| その他（索敵値なし） | **0** | 0 | 全サイト |

### 4-4. 分岐点係数（主要マップ）

| Cn値 | 対象マップ | 出典 |
|------|-----------|------|
| 1 | 2-5 | KCWIKI |
| 2 | 4-5, 5-2, 5-4, 5-5 | KCWIKI |
| 3 | 1-6, 6-2, 6-3, 6-5 | KCWIKI |
| 4 | 3-5, 6-1, 7-2, 7-5 | KCWIKI |

**出典差異**: JP Wikiが各海域の分岐点係数を最も網羅的に記載。英語Wikiは主要マップのみ。

### 4-5. 司令部レベル係数（Q）

| マップ | Q値 | 出典 |
|--------|-----|------|
| 5-2, 6-3 | 0.35 | KCWIKI |
| その他 | 0.4 | KCWIKI |

### 4-6. 出撃艦数補正（M）

| 出撃艦数 | M値 | 出典 |
|---------|-----|------|
| 6隻 | 0 | 全サイト |
| 5隻 | +1 | 全サイト |
| 4隻 | +2 | 全サイト |
| 7隻（遊撃部隊） | -1 | KCWIKI |

### 4-7. 素索敵の計算

```
素索敵 = 艦の索敵ステータス - 装備による索敵加算値
各艦の寄与 = sqrt(素索敵)
```

## 5. フィット砲補正

### 5-1. 概要

艦種ごとに「合う砲」「合わない砲」があり、命中率にボーナス/ペナルティが付く。

```
フィット命中補正 = フィット値 x sqrt(同種砲の搭載数)
```

### 5-2. 戦艦系フィット砲（代表例）

#### 金剛型

| 砲の口径 | 補正値(1本目安) | 出典 |
|---------|----------------|------|
| 35.6cm砲系 | +4～+7 | JP Wiki |
| 38cm砲系 | +2～+3 | JP Wiki |
| 41cm砲系 | -3～-5 | JP Wiki |
| 46cm砲系 | -7～-10 | JP Wiki |

#### 長門型

| 砲の口径 | 補正値(1本目安) | 出典 |
|---------|----------------|------|
| 41cm砲系 | +4～+5(フィット) | JP Wiki |
| 46cm砲系 | -3～-5 | JP Wiki |

#### 大和型

| 砲の口径 | 補正値(1本目安) | 出典 |
|---------|----------------|------|
| 46cm砲系 | +2～+5(フィット) | JP Wiki |
| 51cm砲系 | 0～+2 | JP Wiki |

### 5-3. 過積載ペナルティ

- 過積載砲を複数積むほどペナルティ増大（sqrt(n) に比例）
- 主に昼砲撃戦の命中率低下。夜戦でも影響するが昼戦ほど顕著ではない

**出典差異**: JP Wikiが最も詳細なテーブル。英語Wiki間で+-1～2の差異あり。

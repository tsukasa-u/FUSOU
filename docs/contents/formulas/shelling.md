# 昼戦砲撃・雷撃・対潜・装甲・ダメージ計算

## 1. 基本攻撃力

### 1-1. 昼戦砲撃（水上艦・空母以外）

```
基本攻撃力 = 火力 + 改修強化値 + 連合艦隊補正 + 5
```
- **出典**: JP Wiki / Fandom / KCWIKI 全一致

### 1-2. 昼戦砲撃（空母系）

```
基本攻撃力 = floor((火力 + 雷装 + floor(爆装 x 1.3) + 改修強化値) x 1.5) + 55
```
- 爆装を1.3倍→切り捨て→他ステータスと合算→1.5倍→切り捨て→+55
- **出典**: 3サイト完全一致。表記差のみ（`int` vs `floor` vs `[]`）

### 1-3. 雷撃戦

```
基本攻撃力 = 雷装 + 改修強化値 + 連合艦隊補正 + 5
```
- **出典**: 3サイト一致

### 1-4. 対潜攻撃

```
基本攻撃力 = (2 x sqrt(素対潜) + 1.5 x 装備対潜合計 + 改修補正 + 対潜シナジー補正) x シナジー倍率
```
- 定数: 爆雷/ソナー攻撃時 = **13**、航空対潜時 = **8**
- ソナー改修補正: `0.75 x sqrt(★)`
- 爆雷改修補正: `0.75 x sqrt(★)`（KCWIKI記載では `1.0 x sqrt(★)`）
- **出典**: 3サイト概ね一致。定数の攻撃種別による使い分けはKCWIKI/Fandomが詳細

### 1-5. 航空攻撃（空母砲撃戦）

```
基本攻撃力 = 種別定数 x ((爆装 or 雷装 + 改修補正 + ボーナス) x sqrt(搭載数) + 25 + 連合艦隊補正) + 噴式補正
```
- 種別定数: 雷撃機=1.0（推定）、爆撃機=1.0（推定）
- **出典**: KCWIKI が最も詳細

### 対潜シナジー補正

| 組み合わせ | シナジー倍率 | 出典 |
|-----------|-------------|------|
| ソナー + 爆雷投射機 | x1.15 | 全サイト共通 |
| ソナー + 爆雷 | x1.15 | Fandom/KCWIKI |
| 爆雷投射機 + 爆雷 | x1.10 | Fandom/KCWIKI |
| 3点セット（ソナー+投射機+爆雷） | x1.15 x 1.10 ≒ x1.265 | KCWIKI |

- 「爆雷」と「爆雷投射機」は**別カテゴリ**として厳密に区別される
- JP Wikiは基本的な1.15倍の記述、KCWIKI/Fandomが3点セットの乗算処理を詳述

## 2. キャップ前補正

計算順序:
```
キャップ前攻撃力 = 基本攻撃力 x 交戦形態補正 x 陣形補正 x 損傷補正 + 加算系補正
```

### 2-1. 交戦形態補正

| 交戦形態 | 補正値 | 出典 |
|---------|--------|------|
| 同航戦 | 1.0 | 全サイト一致 |
| 反航戦 | 0.8 | 全サイト一致 |
| T字有利 | 1.2 | 全サイト一致 |
| T字不利 | 0.6 | 全サイト一致 |

### 2-2. 陣形補正（通常艦隊）

| 陣形 | 砲撃 | 雷撃 | 対潜 | 出典 |
|------|------|------|------|------|
| 単縦陣 | 1.0 | 1.0 | 0.6 | 全サイト一致 |
| 複縦陣 | 0.8 | 0.8 | 0.8 | 全サイト一致 |
| 輪形陣 | 0.7 | 0.7 | 1.2 | 全サイト一致 |
| 梯形陣 | 0.75 | 0.6 | 1.0-1.1 | **差異あり** |
| 単横陣 | 0.6 | 0.6 | 1.3 | 全サイト一致 |
| 警戒陣(主力) | 0.5 | - | - | JP Wiki/Fandom |
| 警戒陣(警戒) | 1.0 | - | - | JP Wiki/Fandom |

**差異**: 梯形陣の対潜補正について Fandom=1.0、KCWIKI=1.1。最新検証では1.1が優勢。

### 2-3. 連合艦隊陣形補正

| 陣形 | 砲撃補正 | 出典 |
|------|---------|------|
| 第一警戒航行序列（対潜） | 0.8 | 全サイト一致 |
| 第二警戒航行序列（前方） | 1.0 | 全サイト一致 |
| 第三警戒航行序列（輪形） | 0.7 | 全サイト一致 |
| 第四警戒航行序列（戦闘） | 1.1 | 全サイト一致 |

### 2-4. 損傷補正

| 損傷状態 | 砲撃 | 雷撃 | 出典 |
|---------|------|------|------|
| 小破以下 | 1.0 | 1.0 | 全サイト一致 |
| 中破 | 0.7 | 0.8 | 全サイト一致 |
| 大破 | 0.4 | 攻撃不可 | 全サイト一致 |

- 空母系: 中破以上で砲撃不可（装甲空母は中破でも攻撃可能、補正0.7）

### 2-5. 連合艦隊攻撃力補正（加算）

| 連合艦隊種別 | 第1艦隊 | 第2艦隊 | 出典 |
|-------------|---------|---------|------|
| 機動部隊 | +2 | +10 | 全サイト一致 |
| 水上部隊 | +10 | -5 | 全サイト一致 |
| 輸送部隊 | -5 | +10 | 全サイト一致 |

### 2-6. 軽巡軽量砲補正（加算）

```
軽巡軽量砲補正 = sqrt(単装砲の数) x 2 + sqrt(連装砲の数) x 3
```
- 対象: 14cm単装砲、15.2cm連装砲系列等
- **出典**: JP Wikiが最も詳細

## 3. キャップ処理

### キャップ値

| 攻撃種別 | キャップ値 | 出典 |
|---------|-----------|------|
| 昼戦砲撃 | **360** | KCWIKI（最新） |
| 雷撃戦 | **180** | KCWIKI |
| 対潜攻撃 | **180** | KCWIKI |
| 航空攻撃 | **180** | KCWIKI |

### キャップ処理計算

```
if (キャップ前攻撃力 <= キャップ値):
    キャップ後攻撃力 = キャップ前攻撃力
else:
    キャップ後攻撃力 = キャップ値 + sqrt(キャップ前攻撃力 - キャップ値)
```
- ソフトキャップ方式。超過分は平方根で圧縮
- **出典**: 3サイト完全一致

### キャップ値の歴史的変遷

| 時期 | 昼戦砲撃 | 夜戦 | 出典 |
|------|---------|------|------|
| 初期 | 150 | 300 | JP Wiki |
| 2021年頃 | 180 | 300 | JP Wiki |
| 2023年頃 | 220 | 360 | JP Wiki/KCWIKI |
| 現在 | 360 | 360 | KCWIKI |

## 4. キャップ後補正

```
最終攻撃力 = floor(キャップ後攻撃力)
    x 弾着補正 x 戦爆連合CI補正 x 徹甲弾補正 x クリティカル補正 x 特殊攻撃補正
```

### 4-1. 弾着観測射撃倍率

| 攻撃種別 | 倍率 | 出典 |
|---------|------|------|
| 連撃（Double Attack） | 1.2 | 全サイト一致 |
| 主砲/副砲CI | 1.1 | 全サイト一致 |
| 副砲/電探CI | 1.2 | 全サイト一致 |
| 副砲/徹甲弾CI | 1.3 | 全サイト一致 |
| 主砲/主砲CI | 1.5 | 全サイト一致 |
| 主砲/瑞雲CI | 1.35 | KCWIKI |
| 主砲/彗星CI | 1.3 | KCWIKI |

### 4-2. 徹甲弾補正（対特定艦種）

| 装備組み合わせ | 対戦艦 | 対重巡 | 出典 |
|--------------|--------|--------|------|
| 主砲 + 徹甲弾 | 1.08 | 1.08 | 全サイト一致 |
| 主砲 + 徹甲弾 + 電探 | 1.10 | 1.10 | KCWIKI |
| 主砲 + 徹甲弾 + 副砲 | 1.15 | 1.15 | 全サイト一致 |

### 4-3. クリティカル補正

```
クリティカル時攻撃力 = キャップ後攻撃力 x 1.5 x 熟練度クリティカル補正
```
- 基本クリティカル倍率: **1.5**（全サイト一致）
- 熟練度クリティカル補正: 空母の搭載スロット熟練度に依存（最大+0.2程度）
- KCWIKI が内部計算式を最も詳細に記載

### 4-4. 特殊砲撃（タッチ系）

| 特殊攻撃 | 攻撃者 | 倍率 | 出典 |
|---------|--------|------|------|
| Nelson Touch (1撃目) | Nelson | 2.0 | 全サイト一致 |
| Nelson Touch (2撃目) | 3番艦 | 2.0 | 全サイト一致 |
| Nelson Touch (3撃目) | 5番艦 | 2.0 | 全サイト一致 |
| 長門改二特殊 (1撃目) | 長門 | 1.4 | 全サイト一致 |
| 長門改二特殊 (2撃目) | 2番艦 | 1.2 | 全サイト一致 |
| 陸奥改二特殊 (1撃目) | 陸奥 | 1.4 | 全サイト一致 |
| 陸奥改二特殊 (2撃目) | 2番艦 | 1.2 | 全サイト一致 |

## 5. 装甲・最終ダメージ

### 5-1. 防御力計算

```
防御力 = 装甲 x 0.7 + 装甲乱数 x 0.6
```
- 装甲乱数: `[0, 装甲値-1]` の整数乱数（KCWIKI）
- つまり防御力は `装甲 x 0.7` ～ `装甲 x 1.3` の範囲
- **表記差異**: JP Wikiは「装甲の0.7～1.3倍の乱数」、KCWIKIは分解した形式。数学的に同値

### 5-2. 最終ダメージ

```
最終ダメージ = floor((最終攻撃力 - 防御力) x 弾薬量補正)
```

### 5-3. 割合ダメージ（カスダメ）

攻撃力が防御力を下回る場合:
```
割合ダメージ = floor(現在HP x 0.06 + random[0, 現在HP-1] x 0.08)
```
- ダメージ範囲: 現在HPの約 **6%～14%**
- 0ダメージはMiss（回避）時のみ

### 5-4. 弾薬量補正

```
残弾薬率 >= 50%: 弾薬量補正 = 1.0
残弾薬率 < 50%:  弾薬量補正 = 残弾薬率 x 2
```

| 残弾薬率 | 補正 |
|---------|------|
| 50%以上 | 1.0 |
| 40% | 0.8 |
| 30% | 0.6 |
| 20% | 0.4 |
| 10% | 0.2 |
| 0% | 0.0 |

**出典**: 全サイト完全一致

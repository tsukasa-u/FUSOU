## 1. 疲労度（cond値・モラル）

### 1-1. 基本仕様

- cond値の範囲: **0～100**
- 通常時（未出撃）のcond値: **49**（ソフトキャップ）
- キラキラ状態: **cond >= 50**
- 疲労状態: **cond < 40**

### 1-2. 自然回復

|------|--------|------|
| 港で待機（3分ごと） | **+3** | 全サイト一致 |

| 給糧艦（APF）搭載 | +1/15分、上限54 | KCWIKI |
| 給糧艦改（APF Kai）搭載 | +2/15分、上限54 | KCWIKI |

## 2. 経験値

### 2-1. 出撃時の艦娘経験値

```
取得経験値 = floor(基本経験値 x MVP補正 x 旗艦補正 x 勝敗補正)
```

| 補正項目 | 値 | 出典 |
|---------|-----|------|
| MVP補正 | MVP: x2.0 / 非MVP: x1.0 | 全サイト一致 |
| 旗艦補正 | 旗艦: x1.5 / 非旗艦: x1.0 | 全サイト一致 |
| 勝敗補正（S勝利） | x1.2 | 全サイト一致 |
| 勝敗補正（A/B勝利） | x1.0 | 全サイト一致 |
| 勝敗補正（C敗北） | x0.8 | KCWIKI |
| 勝敗補正（D敗北） | x0.7 | KCWIKI |
| 勝敗補正（E敗北） | x0.5 | KCWIKI |

**最大倍率**: 旗艦 x MVP x S勝利 = `1.5 x 2.0 x 1.2 = **3.6倍**`

- 基本経験値はマップ・ノードごとに固定値が設定されている
- **出典**: 全サイト一致

### 2-2. 演習時の艦娘経験値

3段階の計算処理:

#### (1) キャップ前
```
Exp_pre = floor(相手旗艦の累積経験値 / 100 + 相手2番艦の累積経験値 / 300) + random(0, 3)
```

#### (2) キャップ処理
```
if Exp_pre > 500:
    Exp_cap = 500 + floor(sqrt(Exp_pre - 500))
else:
    Exp_cap = Exp_pre
```
- キャップ値: **500**

#### (3) キャップ後
```
最終経験値 = floor(Exp_cap x MVP補正 x 旗艦補正 x 勝敗補正 x 練習巡洋艦補正)
```

#### 練習巡洋艦ボーナス

| 条件 | 倍率 | 出典 |
|------|------|------|
| 練巡1隻（低Lv） | x1.03～1.05 | KCWIKI |
| 練巡1隻（高Lv） | x1.10～1.15 | KCWIKI |
| 練巡2隻 | x1.20～1.25 | KCWIKI |

**出典**: KCWIKI が最も詳細。Fandom も概要を記載。

### 2-3. 提督（HQ）経験値

#### 出撃時
- マップ・ノード・勝敗ランクで固定値
- S勝利: 基本値の100%
- A勝利: 基本値の80%（非ボス）
- B勝利: 基本値の50%（非ボス）
- 提督経験値上限: **554,400,000**

#### 演習時
| レベル差 | S勝利 | A勝利 | B勝利 | 出典 |
|---------|-------|-------|-------|------|
| 大差（+5以上） | 160 | 120 | 96 | KCWIKI |
| 小差（+-5未満） | 中間値 | 中間値 | 中間値 | KCWIKI |
| 大差（-5以下） | 20 | 15 | 12 | KCWIKI |

## 3. 入渠（修理）

### 3-1. 入渠時間

```
入渠時間(秒) = HP損失量 x 修理時間基礎値 x 艦種係数 + 30
```

#### 修理時間基礎値

```
Level <= 11: 基礎値 = Level x 10
Level >= 12: 基礎値 = Level x 5 + floor(sqrt(Level - 11)) x 10 + 50
```

#### 艦種係数

| 艦種 | 係数 | 出典 |
|------|------|------|
| 潜水艦 | 0.5 | KCWIKI |
| 駆逐艦 | 1.0 | KCWIKI |
| 軽巡洋艦 | 1.0 | KCWIKI |
| 重巡洋艦 | 1.5 | KCWIKI |
| 戦艦 | 2.0 | KCWIKI |
| 正規空母 | 2.0 | KCWIKI |
| 軽空母 | 1.0 | KCWIKI |
| その他 | 1.0～2.0 | KCWIKI |

- 最低でも残り1分未満になると、入渠画面切り替えで即完了可能
- 入渠完了時、condが39以下の場合は **40に自動回復**
- **出典**: KCWIKI

### 3-2. 入渠資源コスト

```
燃料消費 = floor(基本燃料 x HP損失量 x 0.032)
鋼材消費 = floor(基本燃料 x HP損失量 x 0.06)
```

- **基本燃料**: 艦娘の最大燃料消費量（改造段階で変動）
- **出典**: KCWIKI

### 3-3. 泊地修理（明石修理）

- 明石（修理艦）を旗艦配置して20分ごとに修理
- 修理対象: 2番艦～（修理能力に応じて修理範囲が変動）
- 修理量: 小破程度まで回復可能
- 修理艦2隻の場合はx0.85の時間短縮効果

## 4. 資源消費（出撃時）

### 4-1. 戦闘ノードでの消費

| 活動 | 燃料消費 | 弾薬消費 | 出典 |
|------|---------|---------|------|
| 昼戦（通常ノード） | 最大燃料の **20%** | 最大弾薬の **20%** | JP Wiki/全サイト |
| 夜戦追加 | + **10%** | + **10%** | JP Wiki/全サイト |
| ボス戦（昼＋夜） | 合計 **30%** | 合計 **30%** | 推定 |
| 航空戦マス（空襲） | **8%** | **4%** | KCWIKI |
| 渦潮（弾薬） | 弾薬 x(10-30%) 損失 | - | JP Wiki |
| 渦潮（燃料） | 燃料 x(10-30%) 損失 | - | JP Wiki |

**注**: 渦潮の損失は電探搭載数に応じて軽減される。

### 4-2. 潜水艦の燃料消費

| 活動 | 消費率 | 出典 |
|------|--------|------|
| 通常戦闘 | **8%** | JP Wiki |
| 夜戦追加 | + **2%** | JP Wiki（推定） |

### 4-3. 弾薬ペナルティ

残弾薬率と攻撃力補正については shelling.md「5-4. 弾薬量補正」を参照。

```
残弾薬率 >= 50%: 攻撃力補正 = 1.0
残弾薬率 < 50%:  攻撃力補正 = 残弾薬率 x 2
```

### 4-4. 補給コスト

```
燃料補給 = 最大燃料 - 現在燃料
弾薬補給 = 最大弾薬 - 現在弾薬
```
- ボーキサイト: 搭載機数の損失分に応じて消費（艦載機1機あたり5ボーキサイト）
- **出典**: 全サイト一致

## 5. 戦闘勝利判定

### 5-1. 勝敗条件

| ランク | 条件 | 出典 |
|--------|------|------|
| S勝利 | 敵全滅 | 全サイト一致 |
| A勝利 | 敵旗艦撃沈 + 敵の総HP損失 >= 2.5倍（味方損失比） | KCWIKI |
| B勝利 | 敵旗艦撃沈 or 敵の総HPゲージ50%以上削り | KCWIKI |
| C敗北 | 味方旗艦大破 or 味方の損害が大きい | KCWIKI |
| D敗北 | 上記いずれにも該当しない敗北 | KCWIKI |
| E敗北 | 味方全滅 | KCWIKI |

**注**: 勝利判定の正確な数値閾値については検証が完全には済んでいない部分がある。

### 5-2. MVP判定

```
MVP = 昼戦ダメージ + 夜戦ダメージが最大の艦
```
- 同値の場合は **艦隊内の配置順が若い艦** が優先
- **出典**: 全サイト一致

## 6. 遠征

### 6-1. 遠征成功条件

- 艦隊の編成条件（艦種・数）を満たすこと
- 旗艦のレベルが一定以上
- 一部の遠征はキラキラ状態で **大成功** になりやすい

### 6-2. 大成功条件

```
大成功率の推定条件:
- キラキラ艦が4隻以上で大成功率が大幅に上昇
- 旗艦がキラキラで追加ボーナス
```
- 大成功時: 資源獲得量が **x1.5倍**
- **出典**: JP Wiki/Fandom（正確な発動率は検証中）

### 6-3. 遠征の資源消費

```
燃料消費 = 各艦の最大燃料 x 消費率（遠征ごとに異なる）
弾薬消費 = 各艦の最大弾薬 x 消費率（遠征ごとに異なる）
```
- 一般的な消費率: 燃料 **30～50%**、弾薬 **0～50%**
- 遠征ごとに個別設定

# 検証済み計算式 総合ドキュメント

## 概要

本ドキュメント群は、艦これ（艦隊これくしょん）において有志の検証により発見・推定された各種計算式を、
複数の検証サイト・Wikiの情報を突き合わせて包括的にまとめたものである。

## 注意事項

- 本文書の情報は **2025年1月時点** までの検証結果および各Wikiの記載に基づく
- 艦これは定期アップデートでキャップ値や補正値が変更されるため、最新の正確な値は各ソースサイトを直接確認すること
- 各計算式は**公式に公開されたものではなく**、有志の検証（統計的推定・データマイニング）に基づく
- サイト間で値が異なる場合はその旨を明記している

## 全体計算フロー（共通ダメージ計算）

```
[Step 1] 基本攻撃力の算出（攻撃種別で異なる）
    |
[Step 2] キャップ前補正の適用（乗算）
    +-- x 交戦形態補正
    +-- x 陣形補正
    +-- x 損傷補正
    +-- + 加算系補正（軽巡軽量砲補正等）
    |
[Step 3] キャップ処理
    +-- キャップ値以下: そのまま
    +-- キャップ値超過: Cap + sqrt(攻撃力 - Cap)
    |
[Step 4] キャップ後補正の適用（乗算）
    +-- x 弾着観測射撃/戦爆連合CI倍率
    +-- x 徹甲弾補正
    +-- x クリティカル補正 (1.5 x 熟練度補正)
    +-- x 特殊攻撃補正
    |
[Step 5] 最終ダメージ
    +-- 防御力 = 装甲 x 0.7 + 装甲乱数 x 0.6
    +-- ダメージ = floor((攻撃力 - 防御力) x 弾薬量補正)
    +-- 防御力が上回る場合: 割合ダメージ（カスダメ）
```

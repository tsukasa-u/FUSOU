## 1. 支援艦隊

### 1-1. 支援の種類と発動条件

| 支援種別 | 発動条件（編成） | 出典 |
|---------|----------------|------|
| 砲撃支援 | 戦艦系2隻以上（空母系が軽空母1隻以下） | 全サイト |
| 航空支援 | 空母系2隻以上 | 全サイト |
| 対潜支援哨戒 | 軽空母1隻以上+海防艦/駆逐艦等 | 全サイト |
| 雷撃支援 | 上記いずれにも該当しない場合 | 全サイト |

### 1-2. 砲撃支援の攻撃力

```
基本攻撃力 = 4 + 火力 + 改修火力補正
```
- キャップ値: **170**
- **出典**: JP Wiki/KCWIKI

### 1-3. 航空支援の攻撃力

```
基本攻撃力 = 55 + floor(1.3 x (爆装 or 雷装) x sqrt(搭載数)) + 各種補正
```
- キャップ値: **170**
- **出典**: JP Wiki/KCWIKI

### 1-4. 対潜支援の攻撃力

```
基本攻撃力 = 8 + 装備対潜値合計 x 1.0 + ソナー改修補正
```
- ソナー改修: `0.75 x sqrt(★)`
- **出典**: Fandom/KCWIKI

### 1-5. 雷撃支援の攻撃力

```
基本攻撃力 = 5 + 雷装
```
- キャップ値: **170**
- 命中率が非常に低く実用性が低い

### 1-6. 支援艦隊の命中

```
命中項 = 基本命中 + 装備命中 + 改修命中補正 + レベル補正
```
- レベル補正: `2 x sqrt(Level)` 程度（推定）
- 電探の改修命中: `1.7 x sqrt(★)`（JP Wiki/KCWIKI一致）

**差異**: 電探改修の支援命中への寄与について、JP Wiki/KCWIKIは高い効果を記載。Fandomは記述がやや曖昧。

## 2. 基地航空隊

### 2-1. 基地航空隊の攻撃力

```
基本攻撃力 = 25 + 攻撃力定数 x sqrt(搭載数) + 改修補正
```
- 攻撃力定数: 雷装値（陸攻）or 爆装値（陸爆）
- 陸攻の改修補正: `0.7 x sqrt(★)`
- キャップ値: **220**

### 2-2. 基地航空隊の制空値

#### 出撃時
```
制空値 = floor((対空 + 迎撃 x 1.5 + 対爆 x 2) x sqrt(搭載数) + 熟練度ボーナス)
```

#### 防空時
```
制空値 = floor((対空 + 迎撃 x 2 + 対爆 x 2) x sqrt(搭載数) + 熟練度ボーナス)
```

#### 偵察機による制空補正
-> 03_aerial_combat.md の「1-5. 基地航空隊の制空値」を参照

### 2-3. 陸上攻撃機の行動半径

```
中隊の行動半径 = 中隊内の最小行動半径
```

偵察機・大型飛行艇による延伸:
```
延伸値 = floor(sqrt(偵察機の行動半径 - 最小行動半径))
延伸後行動半径 = 最小行動半径 + min(延伸値, 最大行動半径 - 最小行動半径)
```
- 偵察機の行動半径 > 最小行動半径 の場合のみ延伸
- **出典**: 3サイトとも概ね一致

#### 計算例
- 一式陸攻（半径8）+ 二式大艇（半径12）
  - 最小半径 = 8
  - 延伸値 = floor(sqrt(12-8)) = floor(sqrt(4)) = **2**
  - 延伸後 = 8 + 2 = **10**

### 2-4. 基地航空隊の命中

- 陸攻は比較的高い命中率
- 熟練度が命中率に影響（熟練度MAXで命中ボーナス）
- 陸上偵察機配備による触接相当の命中補正あり
